# Defense slides moved

The PhD defense slide deck used to live here. It now has its own repository,
which is synced to Overleaf:

**https://github.com/gjardad/defense-slides**

Edit the deck there (or in the local clone at `../defense-slides`), push to
`main`, then pull into Overleaf via **Menu → GitHub → Pull GitHub changes**.
Compile with `pdflatex` locally (xelatex hangs on missing Fira fonts) or
XeLaTeX on Overleaf.
